Во второй половине февраля 2026 года вышло обновление 1.2 проекта Just the Browser, который помогает удалить функции ИИ, отчёты телеметрии, спонсорский контент, интеграцию продуктов и другие раздражающие элементы из веб-браузеров для ПК (инструкции для Google Chrome, Microsoft Edge и Firefox). Исходный код решения опубликован на GitHub под лицензией MIT.

Цель проекта Just the Browser — предоставить пользователям «только браузер» и ничего больше, используя скрытые настройки в веб-браузерах, предназначенные для компаний и других организаций. Этот проект включает в себя файлы конфигурации для популярных веб-браузеров, документацию по их установке и изменению, а также простые скрипты установки.

Основные обновления в Just the Browser 1.2:

это обновление не содержит изменений в настройках браузера. Существующим пользователям не нужно обновляться, если настройки работают;

настройки конфигурации: добавлена поддержка Edge Beta, Dev и Canary на macOS;

скрипты установки: исправлены проблемы с дистрибутивами Linux, в которых вместо sudo используется doas.


Например, ранее выяснилось, что Google Chrome без ведома пользователя добавляет на ПК нейросети — браузер скрытно загружает на систему локальную версию Gemini. Бонусом она также скрыто запускается, когда идёт взаимодействие с сервисами Google. Исправить ситуацию можно и вручную. Освобождаем память и запрещаем повторную загрузку Gemini в Chrome:

вписываем в браузер адрес chrome://flags/;

находим параметр Optimization Guide On Device Model и отключаем его;

повторяем с пунктом Prompt API;

находим на диске по пути AppData/Local/Google/Chrome/User Data/OptGuideOnDeviceModel/ файл весом в 4 ГБ и удаляем его вручную.


Ранее вышел открытый проект Winslop для удаления ненужного системного мусора, уменьшения избыточности и восстановления контроля пользователя в Windows 11. Решение упрощает обслуживание системы, оптимизирует производительность и устраняет навязчивые, непрозрачные или избыточные функции, сохраняя при этом целостность ОС Windows. Инструмент работает полностью в автономном режиме, без ИИ, облачных сервисов или телеметрии. Решение подходит для пользователей, которые хотят более чистого, прозрачного и минималистичного интерфейса Windows 11.
