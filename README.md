# Proxyflare GetMD Skill

Скрипт для преобразования веб-страниц в чистый Markdown с использованием Cloudflare Workers.

## Как это работает
Скрипт использует прокси-воркеры для обхода блокировок и библиотеку Trafilatura для извлечения основного контента.

## Использование
Вызывается через созданный alias:
```bash
getmd <URL>
```

Пример:
```bash
getmd https://
```

## Особенности
- Автоматическая маскировка под браузер (User-Agent).
- Сохранение в папку `articles/` с чистым Slug-именем.
- Управление зависимостями через `uv`.
